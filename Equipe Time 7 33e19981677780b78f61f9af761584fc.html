Resumo do Projeto

  • O que vocês compreenderam do que será feito (de 2 a 3 parágrafos):

Este projeto tem como objetivo desenvolver um sistema ERP simplificado para resolver
problemas de organização enfrentados por uma empresa simulada, como controle de
estoque, vendas e financeiro. Foram identificadas diversas lacunas na questão de registros e atualização dos documentos, necessitando assim, de um programa automatizado para reverter esse cenário. Esse programa deverá conter um Banco de Dados atualizado e uma interface simples e intuitiva, visando o seu uso rápido e prático.

Para seu uso, há diversos módulos separando os setores, como por exemplo: Financeiro, Funcionários, Vendas, entre outros. Em cada módulo há propriedades para organizar informações de determinado elemento e haverá uma tela para cada setor. A maioria dos módulos será atualizado automaticamente de acordo com as atividades para evitar erros, porém também haverá alguns campos manuais, como em Estoque para informar caso em produto quebre ou estrague, por exemplo. Com essas medidas, será possível garantir um comércio harmônico e um sistema gerencial confiável, solucionando os desafios e dores de nosso cliente.
